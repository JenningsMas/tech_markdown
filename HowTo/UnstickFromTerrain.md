# How To Escape Terrain Bugs In Deadlock
As Deadlock is in its closed alpha release, getting stuck inside terrain geometry is a common issue. Terrain bugs can be game-ruining, trapping you inside the game's map and preventing you from completing objectives. In this guide you will learn the three main methods to free your character from terrain glitches.

## Prerequisite: Enable Developer Console
To use methods two and three of this guide, you must enable the developer console in settings.

1. Open the pause menu by pressing "esc".
2. Enter the settings menu with the "Settings" button.
3. Navigate to the "Interface" section under the "Game" heading.
4. Enable the "Dev command console" setting by clicking on the toggle switch. Ensure the toggle switch icon is a green check mark.

It is best practice to always leave this setting enabled, as you may need to access the console to address bugs in your matches.

## Method One: The Unstick Button
Valve has helpfully included a dedicated "unstick" button in the pause menu which can help you escape terrain glitches. To use this method, you must not be in combat with an enemy character or NPC troopers. This method is the most reliable and should be attempted first.

1. Open the pasue menu by pressing "esc".
2. Press the "Unstick Hero" button.
![An image of the "unstick" button.](https://github.com/JenningsMas/tech_markdown/blob/main/HowTo/Img/Unstick/unstick_button.jpg)
3. Wait for the unstick command to finish. Do not move your character at this time.

If the command was succesful, you will be teleported back onto the map near where you were stuck.

## Method Two: The Unstick Command
The built-in unstick button sometimes fails to free your character. Should this occur, you can access the developer console and use the "unstick" command manually. To use this method, you must not be in combat with an enemy character or NPC troopers.

1. Open the pause menu by pressing the "esc" key.
2. Open the developer console from the pause menu by pressing the "F7" key.
3. Type "unstick" into the console line and press enter.
![An image of the "unstick" command entered into the console line.](https://github.com/JenningsMas/tech_markdown/blob/main/HowTo/Img/Unstick/unstick_command.jpg)
4. Wait for the unstick command to finish. Do not move your character at this time.

This command will run the "unstick" command, even when the unstick button fails. If the command is successful, you will be teleported back onto the map near where you were stuck.

## Method Three: The Killme Command
As this method will kill your character and initiate a long respawn timer, this method is a last resort should the above commands fail.

1. Open the pause menu by pressing the "esc" key.
2. Open the developer console from the pause menu by pressing the "F7" key.
3. Type "killme" in the console line and press enter.
![An image of the "killme" command entered into the console line.](https://github.com/JenningsMas/tech_markdown/blob/main/HowTo/Img/Unstick/killme_command.jpg)

Your character will die and enter the respawning state. When the respawn timer finishes, you will respawn in your team's base, effectively freeing you from the terrain.

## Conclusion
Terrain bugs are a frustrating and unfortunately common issue in Deadlock. As Valve continues to work on the game, these issues will hopefully become less persistent as bugs are fixed. In the meantime, use the methods above to free your character from terrain glitches and continue your match.

If you encounter a terrain bug, please report it on the [Deadlock Bug Report Forum](https://forums.playdeadlock.com/forums/bug-reports.6/) to inform the developers of the issue.
